LoadBalancer
CSCE412_LoadBalancer

This repository demonstrates the functionality of a load balancer. It allows the user to instantiate a number of servers, and processes requests while randomly generating new ones as well.

Compilation

This project was created using CLion and, thus, can be run using cmake. You can also compile using gcc.
